Это типа кто-то заказал какую-то одежду. 
И клиент приходит в маказин проверить что есть в наличии. 
Вносит свой телефон - Форма 1. и идет проверка

1. Ничего нет - показывается форма 3
2. Что-то есть - показывается форма 2 с двумя списками
   а. Готовые 
   б. Не готовые 

Программа должна изменить размер на весь экран на экранах любого размера
Шрифт используется Comic Sans MS

Таблицы для теста в даны в экселе. Для теста загрузить в SqlLite. использовать FireDac

Customer.TEL - ПримариКей 
invitem."ACCSS" - Это тот же телефон для связи
Invoice.INVNUM - Примари Кей для заказа
Если в invitem.Alteration = False и Invitem.Pickup = true, то этот пункт будет готов к погрузке
